Algorithm Time Complexity

Comparing theoretical (worst-case) performance of max-flow algorithms.

Complexity Analysis

The chart visualizes the worst-case time complexities based on the number of vertices (V) and edges (E).

Ford-Fulkerson: O(E * |f*|)

Depends on the max flow value (|f*|), which can be very large.

Edmonds-Karp: O(V * E²)

Improves on Ford-Fulkerson by using BFS to find shortest augmenting paths.

Push-Relabel (generic): O(V² * E)

A different approach that "pushes" flow locally. With FIFO selection, improves to O(V³).
